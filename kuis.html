Kuis Berpikir Komputasional

1. Apa yang dimaksud dengan berpikir komputasional?

  • Menggunakan komputer untuk menyelesaikan semua masalah
  • Mengembangkan algoritma untuk memecahkan masalah
  • Menghafal kode-kode pemrograman
  • Mengetik cepat di komputer

2. Langkah pertama dalam berpikir komputasional adalah?

  • Mengidentifikasi masalah
  • Mencari solusi yang ada
  • Mengevaluasi hasil
  • Menulis kode program

3. Algoritma adalah?

  • Himpunan instruksi untuk menyelesaikan masalah
  • Kode program yang ditulis dalam bahasa pemrograman
  • Proses debugging program
  • Prosedur untuk menjalankan komputer

4. Contoh algoritma sederhana adalah?

  • Algoritma pengurutan angka
  • Menulis email
  • Menggunakan media sosial
  • Menggambar di komputer

5. Dekompisi adalah?

  • Memecah masalah besar menjadi masalah kecil
  • Mencari pola dalam data
  • Menghapus bug dari program
  • Mengulang kode program

6. Mengapa pola penting dalam berpikir komputasional?

  • Untuk mengidentifikasi kesamaan dalam masalah
  • Untuk menghemat waktu dalam menulis kode
  • Untuk meningkatkan kecepatan komputer
  • Untuk mengurangi ukuran program

7. Abstraksi dalam berpikir komputasional adalah?

  • Mengabaikan detail yang tidak penting
  • Menambahkan detail tambahan ke masalah
  • Menyimpan data dalam variabel
  • Mengoptimalkan program

8. Debugging adalah proses untuk?

  • Menghapus kesalahan dari kode program
  • Menulis dokumentasi program
  • Mengompilasi kode program
  • Menjalankan program

9. Manakah yang bukan termasuk dalam berpikir komputasional?

  • Mencari pola
  • Mengembangkan algoritma
  • Menulis laporan
  • Dekompisi masalah

10. Algoritma yang efisien adalah?

  • Menggunakan sumber daya seminimal mungkin
  • Menghasilkan output yang sama dengan input
  • Menggunakan banyak memori
  • Menggunakan waktu komputasi yang maksimal
